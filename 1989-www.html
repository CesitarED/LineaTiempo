1989 – Creación de la World Wide Web

Descripción

Tim Berners-Lee, científico del CERN en Suiza, propone la World Wide Web integrando tres tecnologías: HTML (lenguaje para crear páginas), HTTP (protocolo para transferir páginas), y URL (direcciones web).

Su objetivo era facilitar el intercambio de información entre científicos. Escribió el primer navegador y servidor web. Importante: La Web es un servicio que funciona sobre Internet, no son lo mismo.
